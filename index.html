Acerca de mi

Ingeniero Electrónico y Data Analyst con conocimientos en estadística y competencias para el análisis y visualización de datos a través de Excel, Python, Power BI y Tableau.

Así mismo, con habilidades para la gestión de bases de datos SQL con MySQL para generar soluciones que permitan la identificación de oportunidades de mejora, tanto a nivel operativo como de toma de decisiones.

Con capacidad de adaptación a entornos cambiantes, aprendizaje continuo y trabajo con equipos interdisciplinarios y multiculturales.

Excel

Power Query, Power Pivot, tablas dinámicas y fórmulas siguen siendo excelentes herramientas para el análisis de datos. Muy útil en el pretratamiento de datos.


Power BI

Mi herramienta de visualización favorita. Me gusta la simpleza de su interfaz, la posibilidad de utilizar DAX y la facilidad con la que se pueden crear dashboards interactivos.

Tableau

Práctica y sencilla. Las opciones que brinda para personalización de informes es amplia. Además, Tableau Public me permite compartir mi trabajo con todo el mundo.

SQL Server

Una de las herramientas indispensables para el manejo de grandes cantidades de datos. Lo utilizo principalmente para estandarización y exploración de datos.

Python

Mi lenguaje preferido. El manejo de librerías como Panda y Numpy facilitan la manipulación de los datos. Cada día aprendo algo nuevo sobre este increíble lenguaje y sus aplicaciones en el análisis de datos.

GitHub

Git me permite la gestión distribuida del trabajo. Es decir, trabajar en equipo sin tener que estar físicamente cerca. Cada coloborador cuenta con su propia copia del repositorio en su equipo. Y cada vez que se realice algún cambio, la modificación se aplica al proyecto en desarrollo.
